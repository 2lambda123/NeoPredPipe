## Getting Started

This tool allows a user to predict neoantigens predicted from vcf files using ANNOVAR and netMHCpan.

## Dependencies

1. Python >= 3.6
   - numpy == 1.14.0
   - h5py == 2.7.0
   - pandas == 0.22.0
   - xlrd == 1.1.0
   - scikit-learn == 0.19.1
2. Further dependencies will be checked in Step 1.
